Oracle Instant Client
Installation on Debian 9.8 amd64


Introduction
============
Oracle Instant Client is a free Oracle database client.
The current version is 12.1.0.2.0, and several versions
back to 10.1.0.5 are available.

1. Download the Oracle Instantclient RPM files from
http://www.oracle.com/technetwork/database/features/instant-client/index-097480.html
Everyone needs either "Basic" or "Basic lite", and
most users will want "SQL*Plus" and the "SDK".


--help.ubuntu.com/community/Oracle%20Instant%20Client
